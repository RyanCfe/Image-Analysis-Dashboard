# Image Analysis Dashboard 

Interactive image processing dashboard built using Streamlit, OpenCV, and NumPy.

## Features
- Transformations (Rotate, Flip, Grayscale)
- Enhancement (Histogram Equalization, Brightness/Contrast)
- Filtering (Mean, Median, Gaussian)
- Edge Detection (Canny, Sobel)
- FFT Frequency Analysis
- ROI (Region of Interest) Analysis

## Tech Stack
- Python
- Streamlit
- OpenCV
- NumPy
- Matplotlib

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
